## ⚙️ Local Development

Clone the repository:

```bash
git clone https://github.com/Kirtan1803/kirtan-portfolio.git
```

Navigate to project directory:

```bash
cd kirtan-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
