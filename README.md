## Local Development

Clone the repository:

```bash
git clone https://github.com/Kirtan1803/kirtan-portfolio.git
```

Navigate into the project:

```bash
cd kirtan-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
